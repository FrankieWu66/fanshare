# FanShare Comms Folder

This is your communication folder. I put things here for you to read while I work on code.

Check this folder regularly. New files appear here when:
- A content draft is ready for your review
- Community strategy needs your input
- Growth experiments need decisions
- Expense changes need approval
- Build updates are ready to post

## How to use

1. Read files in this folder when you have a break
2. Files marked DRAFT need your approval before publishing
3. Files marked ACTION need you to do something
4. Files marked FYI are informational only

## Folder structure

```
comms/
├── READ-ME-FIRST.md          ← you are here
├── community-strategy.md     ← created by /community
├── community-log.md          ← engagement tracking
├── growth-strategy.md        ← created by /growth
├── growth-partners.md        ← partnership targets
├── growth-log.md             ← experiment results
├── content-calendar.md       ← created by /content
├── content-log.md            ← published content archive
└── voice-guide.md            ← FanShare brand voice
```

Files appear as skills generate them. Not all exist yet.
